# Font Library Sources

This document records the official sources and license notes for the local
title font library used by Yuanfang Design.

## Usage Rules

- Font files are downloaded to `assets/fonts/` for local image generation only.
- Font files must not be committed to GitHub.
- Font files must not be sold as standalone font files.
- Font files must not be provided to users as downloadable assets.
- Before production launch, a human must re-review each upstream LICENSE.
